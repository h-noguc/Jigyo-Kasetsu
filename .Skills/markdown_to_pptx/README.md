# シンプルMarkdown→PPTX変換ツール

LLMが生成しやすいシンプルなMarkdown形式から、編集可能なPowerPointファイル（.pptx）を作成するツールです。

## 特徴

- ✅ **シンプルなMarkdown形式**: Marpの特殊構文不要
- ✅ **LLM対応**: AIが生成しやすい標準的なMarkdown記法
- ✅ **編集可能**: PPTXで後から編集・微調整可能
- ✅ **スタイル統一**: ダーク背景・ゴールドカラーの統一されたデザイン

## 使い方

### 基本的な使い方

```bash
python3 simple_markdown_to_pptx.py input.md output.pptx
```

出力ファイル名を省略すると、入力ファイル名から自動生成されます：

```bash
python3 simple_markdown_to_pptx.py input.md
# → input.pptx が生成されます
```

### Markdown形式

#### スライドの区切り

`---`（ハイフン3つ）でスライドを区切ります：

```markdown
# スライド1のタイトル

内容...

---

# スライド2のタイトル

内容...
```

#### タイトルスライド

最初のスライドでH1（`#`）が最初にある場合、自動的にタイトルスライドとして中央揃えされます：

```markdown
# プレゼンテーションタイトル

**サブタイトル**

作成者名
```

#### 見出し

```markdown
# H1見出し（大きなタイトル）
## H2見出し（セクションタイトル）
### H3見出し（サブセクション）
```

#### リスト

```markdown
- 箇条書き項目1
- 箇条書き項目2

1. 番号付き項目1
2. 番号付き項目2
```

#### テーブル

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| データ1 | データ2 | データ3 |
| データ4 | データ5 | データ6 |
```

#### 引用

```markdown
> これは引用文です。
> 複数行にわたる引用も可能です。
```

#### 強調

```markdown
**太字** と *イタリック* が使用できます。
```

#### 1ページにまとめる（single_page）

`<!-- _single_page -->` または `<!-- _no_split -->` をスライドの先頭に記入すると、そのスライドから次の `---` までの内容を1ページにまとめます。フォントサイズが自動調整されます。

```markdown
<!-- _single_page -->

# タイトル

## セクション1
内容...

## セクション2
内容...

---

# 次のスライド（通常）
```

## スタイル設定

スタイルは `simple_markdown_to_pptx.py` の `STYLE` 辞書で一括管理されています：

- 背景色: ダークグレー (#2d2d2d)
- テキスト色: クリーム色 (#f4f1eb)
- 見出し色: ゴールド (#c9a86c)
- フォント: Garamond（フォールバック: Georgia）
- スライドサイズ: 10インチ × 5.625インチ（16:9）

必要に応じてスクリプト内の `STYLE` 辞書を編集してください。

## 高さの自動調整

- **テキストボックスの自動リサイズ**: テキストの量に応じて高さが自動調整されます
- **スライド高さに基づく判定**: スライドの高さ（5.625インチ）に基づいて、ページオーバーフローを動的に判定します
- **single_pageモード**: `<!-- _single_page -->` を使用すると、フォントサイズが自動調整されて1ページに収まります

## LLMでの活用例

### ChatGPT / Claude / Cursor でのプロンプト例

```
以下のトピックでプレゼンテーションスライドを作成してください。
Markdown形式で、`---`でスライドを区切ってください。

トピック: [あなたのトピック]

形式:
- H1見出しでタイトルスライドを作成
- H2見出しで各スライドのタイトル
- リストやテーブルを適切に使用
- `---`でスライドを区切る
```

生成されたMarkdownファイルを `simple_markdown_to_pptx.py` で変換すれば、すぐにPPTXファイルが作成できます。

## Marpとの違い

| 項目 | Marp | このツール |
|------|------|-----------|
| YAMLフロントマター | 必須 | オプション（無視） |
| CSSスタイル | 必要 | スクリプト内で一括設定 |
| 特殊ディレクティブ | `<!-- _class: title -->` など | `<!-- _single_page -->` など |
| 1ページにまとめる | `<!-- _paginate: false -->` | `<!-- _single_page -->` |
| LLM生成のしやすさ | やや複雑 | シンプル |
| 編集のしやすさ | 複雑な構造 | シンプルな構造 |
| テキスト自動リサイズ | なし | あり |

## 注意事項

- コードブロック（\`\`\`）は現在スキップされます
- 画像の挿入は未対応です（PPTXで後から追加してください）
- 複雑なレイアウトはPPTXで後から調整してください
