## チャネル①：パートナー成長型

### 用語の認識合わせ

| 用語 | 意味 |
|------|------|
| **起業支援コーチ** | 「コーチとして独立・起業したい人」を支援するコーチ |
| **起業志望者** | コーチング起業を目指す個人（＝起業支援コーチの顧客） |
| **受講者** | 起業志望者が起業後に獲得するコーチングの受講者（＝エンド顧客） |
| **BPaaS事業者** | 本事業。起業志望者のノンコア業務を型化されたプロセスとして提供する |

### 立ち位置

- 起業支援コーチと協業
- そのコーチの起業志望者に自分たち（BPaaS事業者）のサービスを導入
- 起業志望者の事業立ち上がりを早くする＆ハードルを下げる

### ビジネスモデル

```
起業支援コーチ ──協業──> BPaaS事業者
        │                      │
        │起業志望者紹介          │ノンコア業務をサービスとして提供
        ↓                      ↓
    起業志望者 <────────────────┘
```

- 起業支援コーチが起業志望者を紹介
- BPaaS事業者が起業のノンコア（規約・契約手続き、決済・入金管理、予約調整、事前案内、リマインド、フォローアップ、受講者管理等）をサービスとして提供
- 起業志望者は「自分のスキルを売る」ことだけに集中

### 提供価値

**起業志望者へ**
- 規約・契約・決済の手続きがサービスとして提供されるため、起業の事務ハードルが下がる
- 予約調整・事前案内・リマインド・フォローが自動的に回り、受講者対応に抜け漏れがなくなる
- セッション（＝価値提供）だけに集中でき、立ち上がりが早まる

**起業支援コーチへ**
- 起業志望者の成功率向上→コーチとしての実績向上
- 「オペレーション構築」を教える負担軽減（BPaaSが巻き取るため）
- 紹介フィーによる副収入（オプション）

### アナロジー

| サービス | 構造的負の解決方法 | 参考になる点 |
|---------|------------------|-------------|
| Shopify | ECサイト構築のハードルを極限まで下げる | 「売る」以外を全部巻き取る |
| STORES | 決済・予約・ECを一括提供 | 起業の初期オペレーションを簡素化 |
| freee開業 | 開業届作成を5分で完了 | 「始める」ハードルを劇的に下げた |
| Wix | Webサイト制作を誰でもできるように | 専門知識不要で「見せる」を実現 |
